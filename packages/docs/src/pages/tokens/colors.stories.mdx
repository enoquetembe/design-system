# Enoque UI Design System - Colors

The color palette is a fundamental part of Enoque Design System. It establishes a cohesive and pleasing visual language. This page showcases the colors used in this Design System and provides their corresponding codes for easy reference.

<ColorsGrid />

<br/>
## How to Use

To use these colors in your project, import the color variables from the Enoque UI package and apply them to your styles or components:

```jsx
import { colors } from 'enoque-ui/tokens';

const styles = {
  container: {
    backgroundColor: colors.$gray900,
    color: colors.$gray100,
  },
};
````
